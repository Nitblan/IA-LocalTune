## Project Overview

L0calT8ne is a desktop music player developed entirely in Python as part of the International Baccalaureate (IB) Computer Science Internal Assessment.

The objective of this project is to provide users with a modern and customizable local music platform that offers an experience similar to popular streaming applications while maintaining complete control over personal music libraries.

Unlike traditional streaming services, LocalTune focuses on local audio management, allowing users to organize, import, and play their own music collection without requiring a permanent internet connection.

---

## Problem Statement

Many music platforms depend heavily on subscriptions, internet access, and cloud-based services. Users who prefer owning and managing their own music files often lack modern and user-friendly alternatives.

LocalTune was created to address this issue by providing:

* Offline music playback.
* Local music library management.
* Playlist organization.
* Audio customization options.
* A clean and intuitive graphical interface.
* Independence from subscription-based services.

---

## Features

### Music Playback

* Play local audio files.
* Pause, resume, and stop playback.
* Track navigation controls.

### Library Management

* Import individual music files.
* Import entire folders containing music.
* Organize personal collections.

### User Interface

* Modern graphical interface developed in Python.
* Easy-to-use navigation system.
* Customizable appearance options.

### Personalization

* Visual customization.
* Audio customization.
* User-controlled music organization.

### Offline Operation

* No permanent internet connection required.
* Full functionality using local files.
