Hacer sudo apt update & sudo apt upgrade dependeiendo de la distribucion

L0ocalt8ne es un reproductor de musica hecho 100% en python con GUI intuitiva y amigable para poder escuchar musica sin ningun tipo de restricciones, contando con una personalizacion avanzada tanto de imagen como de sonido.
L0ocalt8ne no requiere ningùn otro tipo de servicio, funciona sin internet y se puede importar musica directamente desde spotify, las instrucciones estan de maneradirecta e intuitivamente, a su vez se puede importar carpetas con musica, acepta todo cualquier tipo de formato de video
